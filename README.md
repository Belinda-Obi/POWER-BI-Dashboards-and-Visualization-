# Project 3:POWER BI Business Intelligence Suite 

# Overview 

In this 3RD project of the my "Data Suite " projects , a collection of end to end data visualization projects have been used to show transformation of  raw buisness data into actionable executive insights.

# Adventure Works Executive Sales Dashboard 

**Focus:** 
Global Sales and Revenue Performance.

**Key Insight:** This dashboard shows a 30,000-foot view of a company health, tracking £123M in revenue accross international territories.

**Technical Highlights:** Built using a direct SQL Server connection and time series forecasting.

# USA Super Store Profitability Report 

**Focus:** Operational Profit & Cost Sold (COGS ) 

**Key Insights:** This report drills down into Profit by Product category. It identifies that while Sales are high (1.92M), the profit margins vary wildly between "Office Supplies and Technology. "

**Technical Insights:** Advanced use of Bar Charts and KPI tiles to track high-volume retail transactions. 

# Discount & Shipping Analysis 

**Focus:** Margin Protection and Efficiency 

**Key Insights:** This visual tracks how Discounts impact the bottom line across different sub-categories like "Paper" and "Computer Peripherals ".

**Technical Insights:** Used to help managers decide if they are giving away too much profit through aggressive discounting
